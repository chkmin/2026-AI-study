## <AI 스터디 1주차 WIL>

### 이번주 스터디 목표
AI 정규 스터디의 1주차인 만큼 AI를 한번 맛본다는 느낌으로 코드 문법보다는 전체적으로 어떤식으로 돌아가는지 큰틀을 위주로 학습해보자!!

### 오늘의 키워드
머신러닝, 생선 분류 문제

**1. 머신러닝**
- 데이터로부터 규칙을 학습하여, 새로운 데이터에 대해 예측이나 판단을 수행하는 알고리즘
- 라이브러리 : pandas(데이터 처리), Numpy(수치 계산), matplotlib(데이터 시각화), scikit-learn(머신러닝 모델 학습) -> 분류, 회귀, 클러스터링
- KNN(k-최근접 이웃 알고리즘): 어떠한 데이터와 거리가 최소인 데이터 k개를 찾아 주변들이 어떤 정답을 지니고 있는지 보고 다수결로 판단!!
- 훈련시(모델 학습시), fit() 함수 사용 // 성능 확인시, score() 함수 사용

**2. 생선 분류 문제**
결과: 그래프상에서 시각적 판단으로는 [도미, 도미, 빙어] 순으로 예측되지만 모델의 예측 결과로는 [1, 0, 0]으로 [도미, 빙어, 빙어]로 예측되어 예측 결과와 그래프의 시각적 판단이 다른 것을 알 수 있다.

### 느낀점
이번 주차 스터디는 첫주차 스터디인 만큼 내용이 적은 것 같고 개념도 나름 쉽게 느껴져서 할만하다는 생각이 들었다. 문법 위주로 공부한다는 마인드보다는 전체적인 알고리즘의 흐름을 본다는 마인드를 갖고 생선 분류 문제를 푸니 암기 부담없이 학습할 수 있었다. 이번 주차에서 내가 얻어가야 할 것은 모두 중요하긴 하지만 KNN(k-최근접 이웃 알고리즘)의 개념과 사용법인 것 같다. 훈련할 때는 fit()을 써서 정답 데이터와 입력 데이터를 학습하고 score()을 통해 성능을 확인하고 predict()를 통해 새로운 데이터를 기계학습을 통해 예측한다라는 개념을 얻어갈 수 있었다.
